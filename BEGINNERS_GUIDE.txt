PANDUAN AWAM: Hermes untuk Teknik Geothermal

Apa ini?
Ini adalah kursus praktis untuk teknisi地热 (geothermal) yang ingin belajar
menggunakan AI Assistant (Hermes Agent) untuk membantu pekerjaan engineering.
Semua contoh menggunakan data panas bumi: sumur produksi, separator,
enthalpy, tekanan, temperatur, dan analisis kelayakan.

Siapa yang cocok?
- Engineer geothermal / reservoir
- Production engineer yang pegang data CSV sumur
- Mahasiswa teknik panas bumi
- Data scientist yang kerja di tim subsurface
- Manajer teknis yang mau evaluasi AI untuk workflow engineering

Apa yang dipelajari?
1. Meminta AI membaca kode dulu sebelum edit (Explore -> Plan -> Code)
2. Memberi instruksi spesifik: file mana, fungsi mana, test mana yang gagal
3. Meminta AI menambahkan test (pengujian) sebelum dipercaya
4. Menyimpan aturan geothermal di file CLAUDE.md agar AI ingat terus
5. Membuat skill (keahlian) geothermal yang bisa dipakai berulang kali
6. Mengirim kode berisiko ke "reviewer AI" (subagent) untuk dicek ulang
7. Menggunakan command shell (terminal) untuk inspeksi data CSV
8. Menggunakan tool thermodynamics (CoolProp / IAPWS) lewat MCP
9. Menjalankan analisis sensitivitas paralel (cukup cepat)

Cara cepat memulai (Quick Start)
1. Buka folder ini di terminal WSL.
2. Install Python dependencies:
     python3 -m pip install -r requirements.txt
3. Jalankan figure generator:
     python3 scripts/generate_course_figures.py
4. Mulai Hermes:
     hermes
5. Masuk ke Exercise 1:
     cd 01_explore_plan_code
6. Baca README.md, lalu coba "before prompt" (vague) lalu "after prompt" (spesifik).
   Perbedaannya adalah inti dari kursus ini.

Aturan Penting (Geothermal Standards)
- Selalu pakai satuan SI (kPa, C, kJ/kg).
- Selalu sebut fase air: subcooled, saturated, two-phase, atau superheated.
- Jangan pernah bikin kurva halus yang menembus dome saturasi tanpa diskontinuitas.
- Wellbore: q > 0, P_wf < P_res, P_wh > 0.
- Setiap perhitungan harus punya: known-value check, monotonicity check, physical bounds check.

Keamanan
Hermes default menanyakan persetujuan sebelum command destruktif.
Untuk batch/CI, gunakan hermes config set approvals.mode manual.
