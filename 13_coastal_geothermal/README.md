# Exercise 13: Coastal Geothermal Systems

## Goal
Evaluate seawater geothermal potential using pygeotoolbox seawater properties.

## Context
Coastal or offshore geothermal uses seawater (S~35 psu) as working fluid.
Property differences from freshwater: higher density, lower thermal conductivity,
higher surface tension.

## Tasks
1. Calculate seawater density at 10 C and 35 psu
2. Compare heat extraction rate with freshwater
3. Calculate pressure drop in pipe (using higher density)
4. Evaluate surface tension effect on boiling
