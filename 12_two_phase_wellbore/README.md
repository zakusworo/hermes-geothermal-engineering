# Exercise 12: Two-Phase Wellbore Flow

## Goal
Model wellbore flow where fluid transitions from single-phase liquid
to two-phase (liquid + vapor) as it rises to the surface.

## Key Concept
When P_wf > P_sat(T_wf): single-phase liquid
When P_wh < P_sat(T_avg): two-phase flow begins

## Tasks
1. Calculate saturation pressure at reservoir T
2. Determine if wellbore flow is single-phase or two-phase
3. Estimate steam fraction at wellhead using enthalpy balance
